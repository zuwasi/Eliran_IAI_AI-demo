# Customer A Requirements

A-REQ-001
The system shall provide encrypted communication channels for all operational traffic between user terminals and ground gateways.

A-REQ-005
The system shall support priority handling for emergency command traffic over routine data traffic.

A-REQ-007
The system shall prevent installation of unsigned software updates on operational terminals and gateway servers.

A-REQ-008
The system shall support remote software update distribution to user terminals over the satellite link.

A-REQ-009
The system shall provide an operator dashboard showing terminal status, link status, alarms, and active service profiles.

A-REQ-013
The system shall retain operational logs for at least 7 years.
