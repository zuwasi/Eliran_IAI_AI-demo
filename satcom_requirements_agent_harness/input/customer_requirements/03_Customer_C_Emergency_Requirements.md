# Customer C Requirements

C-REQ-003
The system shall continue to provide local status, diagnostics, and basic configuration functions when disconnected from central management.

C-REQ-005
The system shall prioritize emergency voice and command messages over video and non-critical data.

C-REQ-006
The system shall provide a simplified operator dashboard suitable for non-specialist users.

C-REQ-010
The system shall support offline storage of mission configuration and synchronization when connectivity is restored.
