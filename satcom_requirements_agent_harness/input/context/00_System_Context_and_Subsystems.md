# System Context and Subsystems

AURORA-SATCOM is a modular satellite communication system intended to provide secure, resilient, high-availability voice, data, telemetry, and command services.

Subsystems:
- User Terminal Subsystem: antenna pointing, modem interface, terminal authentication, local diagnostics, degraded-mode operation.
- Ground Gateway Subsystem: routing, QoS enforcement, gateway redundancy, network edge security.
- Network Management Subsystem: monitoring, configuration, fault management, service provisioning, SLA reporting.
- Security and Cryptography Subsystem: authentication, authorization, encryption, key management, audit, secure boot.
- Mission Planning and Resource Allocation Subsystem: bandwidth reservation, beam capacity planning, priority allocation.
- Telemetry, Tracking, and Control Interface: telemetry ingestion and command forwarding interface.
- Data Analytics and Reporting Subsystem: KPI collection, trend analysis, report generation, data export.
